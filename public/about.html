What is Java?

"Java is a general purpose, high-level programming language first released by Sun Microsystems in 1995. It is designed to have as few implementation dependencies as possible, is free to use, and can run on all platforms. It is concurrent, class-based, and object-oriented." - Webopedia.com

Java Has Three Main Components to it:

1. Java Virtual Machine(JVM)

2. Java Runtime Environment(JRE)

3. Java Development Kit (JDK)
